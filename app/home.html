API de eventos de Angelina Tresca

GET ALL EVENTS

GET https://intense-brushlands-88118.herokuapp.com/api/events

GET EVENTS FILTERING

Todos los filtros pueden ser combinados:

  1. Por fechas 
    GET https://intense-brushlands-88118.herokuapp.com/api/events?startDate=Mon, 01 Aug 2021 00:00:00 UTC&endDate=Mon, 01 Aug 2021 23:59:59 UTC

Son necesarias ambas fechas (startDate y endDate) para que este filtro sea aplicado.

Todas las fechas en esta API deben cumplir con el estándar IETF en inglés) de la forma: Fri, 1 Jan 2010 00:00:00 UTC.

  1. Por nombre 
    GET https://intense-brushlands-88118.herokuapp.com/api/events?name=agos
  2. Por destacado 
    GET https://intense-brushlands-88118.herokuapp.com/api/events?highlighted=true

GET ONE EVENT BY ID

GET https://intense-brushlands-88118.herokuapp.com/api/events/{event_id}

DELETE EVENT BY ID

DELETE https://intense-brushlands-88118.herokuapp.com/api/events/{event_id}

MODIFY EVENT BY ID

PUT https://intense-brushlands-88118.herokuapp.com/api/events/{event_id}
content-type: application/json

{
    "description": "Este evento fue modificado",
    "dates": [
        {
            "datetime": "Mon, 01 Dec 2021 20:30:00 GMT",
            "price": 400
        },
        {
            "datetime": "Mon, 01 Dec 2021 22:30:00 GMT",
            "price": 400
        }
    ],
    "location": "CINE",
    "highlighted": false
}

Los datos que no se envíen no serán modificados. Si se envía el campo dates, reemplazará a todo el array de dates que tenga previamente el evento.

SHARE EVENT BY ID

  1. Compartir en Twitter:

    GET https://intense-brushlands-88118.herokuapp.com/api/events/{event_id}/share/tw

En este momento, este endpoint sólo simula en envío del evento a Twitter.

CREATE NEW EVENT

POST https://intense-brushlands-88118.herokuapp.com/api/events
content-type: application/json

{
    "name": "Evento de agosto",
    "location": "Teatro",
    "description": "Este evento es en agosto",
    "photoUrl": "https://imagekit.androidphoria.com/wp-content/uploads/como-buscar-una-imagen-en-facebook-con-una-foto.jpg",
    "dates": [
        {
            "datetime": "Mon, 01 Aug 2021 20:30:00 GMT",
            "price": 200
        },
        {
            "datetime": "Mon, 01 Aug 2021 21:30:00 GMT",
            "price": 300
        }
    ],
    "highlighted": true
}

El campo highlighted es opcional. Por defecto es false.
